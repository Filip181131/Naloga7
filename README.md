## Glavne funkcionalnosti spletne strani

1. Prikaz domače strani z osnovnimi informacijami.
2. Navigacija do drugih strani (kontakt, informacije).
3. Stran s kontaktnimi podatki (email, telefon, naslov).
4. Stran z opisom podjetja/storitve.
5. Preprost uporabniški vmesnik za ogled vsebine.
